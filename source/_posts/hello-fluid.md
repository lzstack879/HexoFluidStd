这是这个博客的第一篇文章。

我用 Hexo 生成站点，用 Fluid 作为主题。这个组合适合个人博客：结构简单、写作方式接近 Markdown 笔记，部署到 GitHub Pages、服务器或静态托管平台也都很方便。

## 之后可以写什么

- 技术学习笔记
- 项目复盘和踩坑记录
- 读书、生活和长期思考
- 常用工具、脚本和配置备忘

## 常用命令

```bash
npm run server
npm run build
```

本地预览时访问 `http://localhost:4000`，生成后的静态文件会输出到 `public` 目录。
